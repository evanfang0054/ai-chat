# Tool MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为现有 agent-driven chat 增加最小 Tool MVP，支持 `get_current_time` 工具调用、SSE 中间事件透传、执行记录落库和前端状态展示。

**Architecture:** 保持当前 `chat -> agent -> llm` 主链不变，在 API 中新增独立 `tool` 模块作为工具注册与执行边界，由 `agent` 模块通过 LangChainJS 调用工具并产出 tool 事件。前端继续消费单条聊天 SSE 流，在 chat store 中单独维护 `toolExecutions` 状态，并在聊天页渲染简单工具执行列表。

**Tech Stack:** NestJS, LangChainJS, Prisma, PostgreSQL, React, Vite, Zustand, TypeScript, Jest, Vitest

- [ ] **Step 3: 写 ToolService 的最小注册与执行逻辑**

```ts
@Injectable()
export class ToolService {
  private readonly tools = new Map([[getCurrentTimeTool.name, getCurrentTimeTool]]);

  constructor(private readonly prisma: PrismaService) {}

  listDefinitions() {
    return [...this.tools.values()];
  }

  async executeTool(name: string, input: string, context: ToolExecutionContext) {
    const tool = this.tools.get(name);
    if (!tool) {
      throw new Error(`Unknown tool: ${name}`);
    }

    const execution = await this.prisma.toolExecution.create({
      data: {
        sessionId: context.sessionId,
        toolName: name,
        status: 'RUNNING',
        input
      }
    });

    try {
      const output = await tool.execute({}, context);
      const updated = await this.prisma.toolExecution.update({
        where: { id: execution.id },
        data: {
          status: 'SUCCEEDED',
          output: JSON.stringify(output),
          finishedAt: new Date()
        }
      });

      return { execution: updated, outputText: JSON.stringify(output) };
    } catch (error) {
      const updated = await this.prisma.toolExecution.update({
        where: { id: execution.id },
        data: {
          status: 'FAILED',
          errorMessage: error instanceof Error ? error.message : 'Tool execution failed',
          finishedAt: new Date()
        }
      });

      throw { execution: updated };
    }
  }
}
```

- [ ] **Step 3: 先实现最小规则触发版 tool 流，打通闭环**

```ts
async *streamChatReply(input: StreamChatReplyInput): AsyncGenerator<AgentStreamEvent> {
  const lowerPrompt = input.prompt.toLowerCase();
  const shouldUseTimeTool = lowerPrompt.includes('time') || input.prompt.includes('时间');

  if (shouldUseTimeTool) {
    const runningExecution = {
      id: 'pending',
      sessionId: input.sessionId,
      toolName: 'get_current_time',
      status: 'RUNNING' as const,
      input: '{}',
      output: null,
      errorMessage: null,
      startedAt: new Date(),
      finishedAt: null
    };

    yield { type: 'tool_started', execution: this.toExecutionSummary(runningExecution) };

    try {
      const result = await this.toolService.executeTool('get_current_time', '{}', {
        sessionId: input.sessionId,
        userId: input.userId
      });

      yield { type: 'tool_completed', execution: this.toExecutionSummary(result.execution) };

      const response = `Current time: ${result.outputText}`;
      yield { type: 'text_delta', delta: response };
      yield { type: 'run_completed' };
      return;
    } catch (error) {
      if (error && typeof error === 'object' && 'execution' in error) {
        yield { type: 'tool_failed', execution: this.toExecutionSummary((error as { execution: never }).execution) };
      }
      return;
    }
  }

  const model = this.llmService.createChatModel();
  const messages = [
    ...input.history.map((message) => this.toLangChainMessage(message)),
    new HumanMessage(input.prompt)
  ];

  const stream = await model.stream(messages);
  for await (const chunk of stream) {
    const text = this.readChunkText(chunk.content);
    if (text) {
      yield { type: 'text_delta', delta: text };
    }
  }

  yield { type: 'run_completed' };
}
```
